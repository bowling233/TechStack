# Linux

## 各发行版

!!! abstract "这是结合个人使用经历对发行版的评价"

- Ubuntu：个人使用最佳，软件、驱动支持全面，图形界面友好。
    - 尤其是 LTS 版本，稳定性好，软件支持最为全面。
- Debian：服务器最常见，稳定性极佳。内核、软件版本较为保守，个人电脑设备较新可能出现兼容性问题。
