## 概念和基础操作

这些绘图软件中的一些概念和基础操作。

=== "`matplotlib`"

    !!! note "`figure` 对象"

        `matplotlib` 绘图以 Figure 为基础，上面可以放置 Artist。Figure 和 Artist 都可以嵌套（比如 Axes 这个 Artist 上面可以放置很多东西）。举一个例子：

        ```python
        fig = plt.figure() # 创建 Figure
        ax = fig.add_subplot() # 放置 Artist
        figL, figR = fig.subfigures(1, 2) # 创建子 Figure
        figL.add_axes() # 放置 Artist
        fig.savefig() # 保存 Figure
        fig.close() # 关闭 Figure
        ```

        上面的这种调用 Figure 方法的风格称为 OO-style，适用于复杂图像。平常绘制简单图像时，我们更常看到 pyplot-style，由 `pyplot` 在一个全局的图像上操作：

        ```python
        plt.figure()
        plt.plot()
        plt.savefig()
        ```

        - 常用 Figure 选项：
            
            ```text
            layout frameon figsize
            ```

        - 常用的 Artist：

            ```text
            axes subplot colorbar legend suptitle text
            ```
    
    ```python
    import matplotlib.pyplot as plt
    # 笔记本非交互模式
    %matplotlib inline
    # 笔记本交互模式
    %matplotlib notebook
    ```

    - 支持保存为：`pdf png svg`
