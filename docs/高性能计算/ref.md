# 参考文献

一些大学和研究机构已经建成了大规模高性能计算平台，并编写了完善的用户手册。这些大型超算平台的架构和软件配置都有所不同，提供了不同的思路和参考。

- [卢森堡大学：ULHPC Technical Documentation](https://hpc-docs.uni.lu/)
- [普林斯顿大学：Princeton Research Computing](https://researchcomputing.princeton.edu/support/knowledge-base)
- [上海交通大学：交我算平台](https://docs.hpc.sjtu.edu.cn/index.html)

此外，一些院校的 HPC 队伍也在逐步建立各种 Wiki。与上面偏向于应用的用户手册不同，这些 Wiki 中有更多关于 HPC 竞赛、编程相关的内容。

- [HPC Wiki](https://hpcwiki.io)
