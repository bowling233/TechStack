# 如何阅读网络标准

!!! info ""

    - [How to Read an RFC](https://www.ietf.org/blog/how-read-rfc/)
    - [rfc.fyi: Browseable, searchable RFC index](https://rfc.fyi/)

重要信息：

- RFC 头部的 `Status of This Memo` 字段描述了 RFC 的状态，如 `Internet Standard`、`Proposed Standard`、`Informational`、`Experimental`、`Best Current Practice` 等。
- RFC 头部的
    - `Obsolete` 字段描述了该 RFC 是否已经被其他 RFC 取代。
    - `Updates` 字段描述了该 RFC 是否更新了其他 RFC。
    - `Updated by` 字段描述了该 RFC 是否被其他 RFC 更新。
