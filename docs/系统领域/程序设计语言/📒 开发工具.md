# 📒 开发工具

## 编译构建

### CMake

!!! quote

    - [Modern CMake](https://cliutils.gitlab.io/modern-cmake/README.html)

使用：

```shell
cmake -S . -B build
cmake --build build -j -v
cmake --install build
```

指定编译器和构建系统：

```shell
cmake --help
CC=clang CXX=clang++ ...
cmake -G"Ninja"
```

选项：

```shell
cmake -LH
cmake -D...
# 标准选项
-DCMAKE_BUILD_TYPE=Release|Debug
-DCMAKE_INSTALL_PREFIX=...
```
