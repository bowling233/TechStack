??? note "寄存器标识"

    除了 `x0-x31`，还会用其他标识标记寄存器的用途，常见的有：

    - `a0-a7`（`x10-x17`）：函数参数
    - `ra`（`x1`）：返回地址
    - `sp`（`x2`）：栈指针
    - `t0-t6`（`x5-x11`）：临时寄存器，被调不保存
    - `s0-s11`（`x8-x18`）：保存寄存器，如果使用了则需要保存
