## 2-2a, c

$$
\overline{X}\overline{Y} + \overline{X}Y + XY\\
= \overline{X}(\overline{Y}+Y) + XY\\
= \overline{X} + XY\\
= \overline{X} + X \text{ (Additional equation)}
$$

$$
Y+\overline{X}Z+X\overline{Y}\\
= Y(\overline{X}+X) + \overline{X}Z + X\overline{Y}\\
= X + \overline{X}(Y+Z)\\
= X+ Y + Z \text{ (Additional equation)}
$$
