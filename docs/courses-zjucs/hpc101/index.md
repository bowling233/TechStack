## 课程总结、感想与推荐

总的来说， 超算短学期的课程内容十分丰富，是一门硬课。希望选择这门课的同学，能够在上课前主动了解一下「超算是什么？超算能做什么？」等问题。

首先我们来聊聊超算短学期的内容。HPC（High Performance Compute）是高性能计算的意思，而目前，实现高新能计算的最主要方法就是并行计算，本课程实质围绕并行计算展开。作为大一学生，同学们应该很少有并行编程经验，这是一个相对陌生的领域。我们需要学习的内容大纲如下：

以上是个人对 HPC101 课程内容的总结。可以看出，这门课程的内容非常丰富，算得上是一门硬课。如果你对计算机体系结构、并行编程、等领域感兴趣，那么这门课程将会是你的不二之选。学习时，一定要投入足够的时间进行自学，11 天的线下课程绝不是这门课的全部。

接下来聊聊课程感受。这门课给我带来的一大感觉是「充实」：有非常多非常多的知识需要学习，实验中有非常多非常多的方法值得尝试和探索。你投入进去的时间，一定会给你带来相应的收获。其次是「愉快」：上课的是超算队的学长和指导老师，非常好沟通。在独立完成的前提下，你也可以和大佬们交流想法，启发思路。特别是最后的小组作业，我和我的队友们合作得十分愉快（虽然还是被强队薄纱了），是非常难忘的体验。总之，对我来说，「学的开心、收获满满」的，就是一门好课。

最后是对这门课程的推荐。我会推荐：

-  对计算机体系结构、并行编程等领域感兴趣的同学
-  自学能力强、有良好的 C 或 Python 编程基础的同学
-  能够在长周期坚持学习的同学（因为几个 Lab 的周期覆盖了整个暑假，后期很可能会慢慢拖延）

进行学习。这门课程提供了一个非常好的学习平台，你可以在这里学习到很多知识，也可以在这里认识很多志同道合的同学。希望你能够在这里收获满满，学得开心~
