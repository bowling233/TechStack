## Ch1. 引言：递归简论

四条基本法则：

1. 基准情形
2. 不断推进
3. 设计法则：假设所有的递归调用都能正常进行（不关心簿记管理的细节，不必追踪大量递归调用）
4. 合成效益法则：求解一个问题的同一实例时，不在不同的递归调用中做重复性工作（斐波那契数列）。

递归的证明：使用归纳法
