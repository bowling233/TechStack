# 深入理解计算机系统

<!-- prettier-ignore-start -->
!!! abstract
    
    本书的目的是解释所有计算机系统的本质概念，并展示这些概念是如何实实在在地影响应用程序地正确性、性能和实用性的。学习本书，你能够了解如何利用系统知识来编写出更好的程序。

    学习本书的预修要求是：

    - 对 C/C++ 语言有一定了解
    - 对 Linux 系统有一定了解
    - 不需要硬件、机器语言和汇编语言经验

    阅读本书的最好方式是：跟着做，在真正的系统上解决具体的问题，或是编写和运行程序。
<!-- prettier-ignore-end -->

每章作业的答案附在笔记最后一节。详细代码直接连接到仓库，不在笔记中展示。
