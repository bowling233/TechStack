# 数学建模

## 线性模型

### 差分方程

$$
a_{n+1} = ra_n+b
$$

上面形式的差分方程，关键是找不动点。解 $a = ra + b$ 得到不动点 $a = \frac{b}{1-r}$。该系统的解是 $a_k = r^kc + \frac{b}{1-r}$。

## 基本概念

决策变量、目标函数、约束条件

## 线性规划

-   求解：
    -   转化为 Matlab 标准型：目标函数最小值、约束条件小于等于或等号

```matlab

```
 
## 非线性规划

-   求解：

```matlab
[x, fval] = fmincon(fun, x0, A, b, Aeq, beq, lb, ub, nonlcon)
```
